# Stress-Parameter-Sensitivity

## 阿拉斯加地区强震库仑应力变化参数敏感性分析

技术栈： 库仑应力变化理论（Coulomb Failure Criterion）、误差传播推导、线性化建模、滑动模型（Slip Model）、参数敏感性统计分析

具体工作：推导库仑应力变化（ΔCFF）的误差传播公式，完成从基础方程到线性化形式的完整数学展开；获取并处理阿拉斯加地区地震相关数据，构建6种不同滑动模型；对比分析6个模型参数（含摩擦系数、几何参数等）对库仑应力变化的方差贡献，区分第一参数与其余五参数的影响差异；计算后五参数在总方差中的平均占比，评估各参数相对重要性。

量化成果：明确第一参数（经验参数B除外）方差贡献在6个模型中保持一致，后五参数方差贡献存在显著差异；给出后五参数平均占比表（共6组模型），其中φ参数平均占比最高（约0.34–0.40），其余参数占比分别为：μ约0.07–0.08，δ相关项约0.14–0.17，λ相关项约0.05，其他项约0.37–0.40，量化了各参数敏感性的相对排序。
